# Skill: request_code_example

**Version**: 1.0.0
**Primary Agent**: ROS2Engineer | SimulationEngineer | IsaacExpert
**Purpose**: Specify requirements for executable code examples with clear inputs, outputs, testing criteria.

## Inputs
- functionality (string)
- language (string: Python | C++ | YAML | XML)
- framework (string: ROS2 | Gazebo | Isaac Sim | Unity)
- inputs_outputs (object)
- dependencies (array)
- testing_criteria (array)
- pedagogical_comments (boolean, default: true)

## Outputs
- Executable code with inline comments
- Setup instructions
- Example usage with expected output
- Unit tests
- Troubleshooting tips

## Success Criteria
- Syntactically correct
- Tested in target environment
- Pedagogical comments explain logic
- Complete setup instructions

## Estimated Time: 1-2 hours per code example
